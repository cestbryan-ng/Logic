# Module de logique

Un moteur de logique propositionnelle léger pour Python. Ce module permet de construire des bases de connaissances complexes et d'effectuer des inférences logiques automatisées.


## Fonctionnalités

- Représentation complète des connecteurs logiques : `NOT`, `AND`, `OR`, `Implication`, `Biconditionnel`.
- Moteur d'inférence basé sur le **Model Checking**.
- Structure orientée objet pour une manipulation facile des symboles et des phrases logiques.

Pour utiliser le module, placez le fichier `logic.py` dans votre dossier de projet et importez-le :
